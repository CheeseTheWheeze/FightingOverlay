# Mobile Client Notes (Scaffold)

## Planned API contract (draft)

* `POST /athletes` – create athlete profile
* `POST /clips` – create clip metadata
* `POST /clips/{clip_id}/upload` – upload raw clip
* `POST /clips/{clip_id}/process` – request processing
* `GET /clips/{clip_id}/status` – polling status

No mobile implementation yet; this is a placeholder for the eventual contract and UX notes.
