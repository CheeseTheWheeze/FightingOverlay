# GrapplingOverlay Quickstart

## Install
1. Download **GrapplingOverlayBootstrap.exe** from the latest GitHub Release.
2. Double-click the file to install/update.
3. Use the **GrapplingOverlay** desktop shortcut for all future launches.

> The bootstrapper handles updates and installation automatically. No zips are required.

## Update
From the Control Center, click **Check for Updates** or **Update Now**.

## Logs
Logs are stored in:
`%LOCALAPPDATA%\GrapplingOverlay\logs`
